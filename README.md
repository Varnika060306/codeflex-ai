# CodeFlex AI 🤖

A voice-powered AI fitness trainer built with Next.js 14, Clerk Auth, Google Gemini, and Vapi.

## Features
- 🎙️ Voice AI onboarding via Vapi
- 🤖 Personalized workout & diet plans generated by Google Gemini
- 🔐 Authentication via Clerk (Google, email)
- 💾 Program storage with localStorage
- ⚡ Cyberpunk UI with Tailwind CSS

## Tech Stack
Next.js 14 · TypeScript · Tailwind CSS · Clerk · Google Gemini API · Vapi
